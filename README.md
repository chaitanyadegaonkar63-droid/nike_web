# Swoosh Brands — Frontend Snapshot


link : https://swoosh-brands.preview.emergentagent.com/


This repository contains the **static frontend** of the Swoosh Brands marketing site:
- `index.html`
- `assets/css/styles.css`
- `assets/js/main.js`
- `assets/images/*` (placeholders)

> ⚠️ This is **frontend only** — server-side code (PHP, database, forms processing) is **not included**.

## Features
- Responsive hero, services, portfolio, about and contact sections
- Mobile-friendly navigation
- Front-end contact form validation (no backend)
- Easy to customize styles and content
